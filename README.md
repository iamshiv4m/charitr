# charitr.com - Premium Domain For Sale

A professional Next.js landing page for selling the premium domain **charitr.com**.

## Features

- **Clean, minimal design** with professional typography
- **Responsive layout** optimized for all devices
- **Contact integration** with WhatsApp and email
- **Inquiry form** for potential buyers
- **Comprehensive SEO optimization** with:
  - Complete meta tags and Open Graph tags
  - Twitter Card support
  - Structured data (JSON-LD) for search engines
  - Automatic sitemap.xml and robots.txt generation
  - PWA manifest for mobile installation
  - Semantic HTML structure
- **Subtle animations** for enhanced user experience
- **Domain value display** ($1,700 USD on GoDaddy)

## Technologies Used

- Next.js 15 with App Router
- TypeScript
- Tailwind CSS
- Lucide React (for icons)
- Inter & Playfair Display fonts
